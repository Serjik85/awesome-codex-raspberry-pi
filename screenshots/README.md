# Screenshots

Place anonymized screenshots in this directory.

Suggested filenames:

```text
remote-mobile.png
codex-running.png
remote-control-start.png
pairing-code.png
```

Before publishing a screenshot, remove or blur:

- Authentication tokens
- Pairing codes
- Email addresses
- Phone numbers
- Public IP addresses
- Private IP addresses, when unnecessary
- Usernames and hostnames, if you prefer anonymity
- Browser tabs containing private projects
